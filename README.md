# Portfolio Project

Monorepo starter based on your previous Capstone structure:

- client: React + Vite frontend
- server: Express backend
- shared: shared constants/types/helpers

## Quick Start

1. Install dependencies from root:

```bash
npm install
```

2. Copy environment variables:

```bash
copy .env.example .env
```

3. Run client + server together:

```bash
npm run dev
```

## Useful Scripts

- npm run dev:client
- npm run dev:server
- npm run build
- npm run start

## Current API Endpoints

- GET / => API status
- GET /api/health => health check
